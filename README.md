# React Scroll Notify

A react wrapper that detects when your component enters the viewport and fires a function passed as prop

## Usage

```js
import Notify from 'react-scrollnotify';
.
.
.
<Notify onvisible={Function}>
  ...
</Notify>
```

## Props

 => onvisible
